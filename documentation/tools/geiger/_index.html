geiger is a standalone application which can connect a Geiger counter to the weather station.

Although this tool is specific to a single manufacturer as their protocol handles multiple models, it should be possible to add additional types of Geiger counter in the future.

It's specifically written for the GQ GMC-320Plus Geiger counter, as that unit supports being connected to the system via USB, and it's protocol is well documented.

A GQ GMC-320plus Geiger counter connected to a Raspberry PI 5B

Setup

Setting up the Geiger counter is pretty simple. Just connect the Micro-USB cable to a USB port on the host as the unit presents a serial port.

Next locate the port it's connected to, usually it's /dev/ttyUSB0 and use that for the -geiger-port parameter.

The last part is to give the Geiger counter an ID within the station and optionally set -rootDir to point to your station's configuration directory.

Syntax

geiger -geiger-id station.geiger -geiger-port /dev/ttyUSB0

Options

OptionDescriptionRequired
-geiger-id idSets the unique id for this sensorRequired
-geiger-port deviceSets the serial port for this sensorRequired
-rootDir pathThe location of configuration files
-geiger-baud baudRateThe baud rate to useDefault: 115200

Example

On my system, the counter appears as /dev/ttyUSB0 and gave it the id of garden.geiger. So to run the service: 

/usr/local/piweather/bin/geiger -geiger-id garden.geiger -geiger-port /dev/ttyUSB0
