weatherscript is a standalone interpreter which, via a simple scripting language similar to C, provides the ability to run scripts against the recorded data.

weatherscript is an implementation of my go-script scripting language and includes the go-anim graphical extensions to support image or timelapse generation.

A primary use of this is in the generation of timelapse videos from a webcam where you can easily overlay data on each frame containing the weather information for that frame.

You can also use it for generating reports from the recorded data.

Syntax

weatherscript script [arguments…]

Options

OptionDescriptionRequired
-rootDir pathThe location of configuration files
