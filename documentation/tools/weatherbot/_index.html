weatherbot is a standalone application which, via a simple template, generate a post of the weather for a specific time period and post it to a Mastodon account.

Syntax

weatherbot -post template

Options

OptionDescriptionRequired
-post templateTemplate to postYes
-host urlurl of weatherdbdefault "http://127.0.0.1:9001"
-rootDir pathThe location of configuration files
-v -testRun in test mode, both parameters are required
