weatherarchive is a utility that can receive messages from weatheringress and archive them to disk.

This component is included within weatheringress so if local archiving is required then archive from within it. This utility is for archiving remotely, potentially from multiple stations to a central location.

Syntax

weatherimport -station station.yaml -import http://127.0.0.1:9001 -dir archiveDir

Options

OptionDescriptionRequired
-rootDir pathThe location of configuration files
-station pathThe station config
-importDatabase URL
-dirDirectory to import data fromOptional
