--- type: "manual" title: "weatherimport" linkTitle: "weatherimport" description: "Import archives recorded by weatherarchive into the database" categories: - weather - piweather.center ---

weatherimport is a utility that can take the archives generated by weatherarchive and import them into the database.

Syntax

weatherimport -station station.yaml -import http://127.0.0.1:9001 -dir archiveDir

Options

OptionDescriptionRequired
-rootDir pathThe location of configuration files
-station pathThe station config
-importDatabase URL
-dirDirectory to import data fromOptional
