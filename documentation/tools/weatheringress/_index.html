weatheringress is a core component which can accept live readings from a sensor, process them and then pass those readings to RabbitMQ where the other components and then use them, primarily weatherdb.

Syntax

weatheringress -rest-port 8081

Options

OptionDescriptionRequired
-rest-port intPort to use for httpYes, default to 8080
-rootDir pathThe location of configuration files
-archive-dir pathLocation to archive inbound requests
-archive-logLog messages to stdout
