The heatindex function calculates the heat index based on the temperature and relative humidity. If the values passed are not of the correct type then this will return an error.

This function is only valid for temperatures above 26.6°C (80°F). As such the feelslike function may be more suitable as it's response will be dependent on the temperature passed to it.

Syntax

weathercalc, weatherdb

heatindex( temperature, humidity )

go

import "github.com/peter-mount/piweather.center/weather/measurement" import "github.com/peter-mount/piweather.center/weather/value" measurement.HeatIndex(temp, humidity value.Value) (value.Value, error)

Definition

The Heat Index can be calculated from just temperature and relative humidity with the following equation with an error of ±1.3°F:

HI = -42.379 + 2.04901523T + 10.14333127R - 0.22475541 T R - 6.83783x 10-3 T2 - 5.481717x10-2 R2 + 1.22874x10-3 T2 R + 8.5282x10-4 T R2 - 1.99x10-6 T2 R2

Where:

  • T is the ambient dry bulb temperature (°F)
  • R the relative humidity as an integer percentage.
