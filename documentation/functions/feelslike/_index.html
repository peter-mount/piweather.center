This function returns the feels like temperature based on the temperature, relative humidity and wind speed.

Syntax

weathercalc, weatherdb

feelslike( temperature, humidity, windSpeed )

go

import "github.com/peter-mount/piweather.center/weather/measurement" import "github.com/peter-mount/piweather.center/weather/value" measurement.FeelsLike(temp, humidity, windSpeed value.Value) (value.Value, error)

Definition

As the Wind Chill and Heat Index functions are only valid across specific ranges, this function will call the appropriate function based on the following conditions:

  • The temperature is above 26.6°C (80°F) then the Heat Index will be returned.
  • The temperature is below 10°C (50°F) and the wind speed is faster than 1.34m/s then the Wind Chill will be returned.
  • If neither of the previous two cases are used then the original temperature is returned.
